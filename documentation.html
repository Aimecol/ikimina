Platform Overview

The Ikimina Digital Platform is a comprehensive web-based management system designed for community-based savings and lending groups in Rwanda. It provides tools for member management, financial tracking, loan processing, and meeting coordination.

Member Management

Complete member lifecycle management with role-based access control.

Financial Operations

Contribution tracking, loan processing, and payment management.

Analytics & Reporting

Comprehensive financial reports and performance analytics.

Mobile Responsive

Fully responsive design optimized for mobile devices.

Installation Guide

Follow these steps to set up the Ikimina Digital Platform on your server.

Terminal 
# Clone the repository
git clone https://github.com/Aimecol/ikimina.git
cd ikimina

# Set up web server (Apache/Nginx)
# Copy files to web root directory
sudo cp -r * /var/www/html/ikimina/

# Set proper permissions
sudo chown -R www-data:www-data /var/www/html/ikimina/
sudo chmod -R 755 /var/www/html/ikimina/

Configuration

Configure the platform settings for your organization.

config.js 
const config = {
    // Organization settings
    organizationName: "Your Ikimina Group",
    currency: "RWF",
    locale: "rw-RW",
    
    // API endpoints
    apiUrl: "https://api.yourdomain.com",
    
    // Mobile Money settings
    mobileMoneyProviders: {
        mtn: {
            enabled: true,
            apiKey: "your-mtn-api-key"
        },
        airtel: {
            enabled: true,
            apiKey: "your-airtel-api-key"
        }
    },
    
    // Email settings
    email: {
        smtp: {
            host: "smtp.yourdomain.com",
            port: 587,
            secure: false,
            auth: {
                user: "noreply@yourdomain.com",
                pass: "your-password"
            }
        }
    }
};

API Reference

Authentication

All API requests require authentication using JWT tokens.

POST /api/auth/login

Authenticate user and receive access token.

Request Body

{
    "email": "user@example.com",
    "password": "userpassword",
    "role": "member"
}

Response

{
    "success": true,
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "user": {
        "id": "12345",
        "email": "user@example.com",
        "role": "member",
        "groupId": "group123"
    },
    "expiresIn": 3600
}

Members API

GET /api/members

Retrieve list of group members.

Headers

Authorization: Bearer {token}
Content-Type: application/json

Query Parameters

Parameter Type Description
page integer Page number (default: 1)
limit integer Items per page (default: 20)
status string Filter by status (active, inactive, pending)

Payments API

POST /api/payments

Process a new payment or contribution.

Request Body

{
    "memberId": "member123",
    "amount": 15000,
    "currency": "RWF",
    "paymentMethod": "mobile_money",
    "provider": "mtn",
    "transactionId": "MM240001234",
    "description": "Monthly contribution"
}
